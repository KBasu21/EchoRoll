# EchoRoll v2 🦉

EchoRoll v2 is a modern Android application designed to transform manual attendance tracking into a seamless, automated, and visually engaging experience for college students. 

Say goodbye to messy spreadsheets and notebook scribbles. EchoRoll empowers you to set attendance goals, map your weekly routine, and instantly view your academic standing through a beautifully crafted, highly intuitive interface.

## Core Features

* **Visual Subject Setup:** Add specific subjects detailing the code, professor, category (Theory/Lab), and custom required attendance percentages.
* **Interactive Dashboard:** View real-time statistics via striking circular progress indicators that calculate classes attended versus missed.
* **Smart Status Alerts:** Instantly translates raw numbers into actionable insights, letting you know if you are safely on track or need to attend more lectures.
* **Dynamic Routine Grid:** A fully integrated weekly timetable that intelligently maps your scheduled classes into seamless, easy-to-read visual blocks.
* **Frictionless UI:** Smooth swipe-to-delete gestures, Material 3 design components, and a custom dark-mode aesthetic.

## Tech Stack & Architecture

This project is built using modern Android development standards:

* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (Material 3)
* **Local Database:** Room (SQLite abstraction)
* **Architecture:** MVVM (Model-View-ViewModel) with Kotlin Coroutines and StateFlow for reactive UI updates.

## Getting Started for Developers

To clone and run this project locally via Android Studio:

1. Clone the repository:
   `git clone https://github.com/YourUsername/EchoRollV2.git`
2. Open the project in **Android Studio**.
3. Allow Gradle to sync the project dependencies.
4. Click the **Run** button to deploy the app to an emulator or physical device.
